# modulo de server

## recomendado

instalar extensiones de:

- prisma
- prettier
- error lens

## ejecucion

1. instalar modulos

```bash
npm i
```

2. setear .env

```bash
code .env
# completar elementos de .env.example
```

3. correr prisma

```bash
npx prisma migrate dev
```

4. iniciar server

```bash
npm run dev
```
